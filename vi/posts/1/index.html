  • Thời gian/ múi giờ
  • Ngôn ngữ / Khu vực
  • Bin / tốc độ
  • Phân vùng ổ cứng

# 📶 Kết nối

  • Wifi
  • Network
  • Bluetooth

# 🖱️ Thiết bị

  • Bảng vẽ
